# Flower e-Shop 🌸

Flower e-Shop is a responsive e-commerce interface designed for an online flower shop. The project focuses on modern UI design, product presentation, and a clean layout to create an engaging shopping experience.

## Features

- Responsive layout for different screen sizes
- Product showcase interface
- Clean and modern UI design
- Mobile-friendly user experience
- Structured layout using reusable design patterns

## Tech Stack

HTML5 | CSS3 | JavaScript | Responsive Design | CSS Grid | Flexbox

## Project Structure

flower-e-Shop/
index.html
style.css
script.js
assets/

- HTML – page structure  
- CSS – styling and layout  
- JavaScript – interactive elements  
- assets – images and visual resources  

## Installation

Clone the repository:

git clone https://github.com/yvejessica/flower-e-Shop

Open the project folder:

cd flower-e-Shop

Open **index.html** in your browser.

## Purpose of the Project

This project was created to practice front-end development, focusing on responsive design, UI layout, and building structured and maintainable interfaces.
